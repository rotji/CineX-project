CineX Wallet Connection Help Guide

Can't Connect Your Wallet? Here's How to Fix It

Quick Fix Checklist

Follow these simple steps to get your wallet connected:

Step 1: Do You Have a Wallet Installed?
- Look for wallet icons in your browser's extension area (top-right corner)
- Common wallets: Xverse, Leather, Asigna
- If you don't see any wallet extensions, go to Step 2

Step 2: Are You Using the Right Browser?

Works on these browsers:
- Google Chrome
- Mozilla Firefox  
- Microsoft Edge
- Brave Browser

Doesn't work on:
- Safari
- Internet Explorer
- Mobile browsers (use mobile apps instead)


Most Common Problems & Easy Fixes

Problem 1: I Don't Have a Wallet

What You'll See:
- Connect Wallet button doesn't work
- Message saying "No wallet found" or similar
- Nothing happens when you click connect

How to Fix:
1. Install a Wallet (Pick One):

   Option A: Xverse Wallet (Recommended)
   - Go to your browser's extension store
   - Search for "Xverse Wallet"  
   - Click "Add to [Browser Name]"
   - Follow setup instructions

   Option B: Leather Wallet
   - Search for "Leather Wallet" in extensions
   - Install and set up

2. After Installing:
   - Refresh the CineX page
   - Try connecting again

Problem 2: I Have a Wallet But It Won't Connect

What You'll See:
- You can see your wallet extension icon
- But CineX still says "no wallet found"
- Connection still doesn't work

How to Fix:
1. Refresh the Page:
   - Press Ctrl+R (Windows) or Cmd+R (Mac)
   - Or click the refresh button in your browser

2. Check if Wallet is Allowed:
   - Right-click your wallet extension icon
   - Look for "Permissions" or "Allow on this site"
   - Make sure it's enabled for CineX

3. Try Restarting Your Browser:
   - Close all browser windows
   - Open browser again and go to CineX

Problem 3: Wallet Connects But Transactions Don't Work

What You'll See:
- Wallet connects successfully
- But you get errors when trying to contribute or create campaigns
- Messages about "wrong network" or similar

How to Fix:
1. Check Your Wallet's Network Setting:
   - Click on your wallet extension
   - Look for "Settings" or gear icon
   - Find "Network" settings
   - Make sure it matches what CineX is using (usually Testnet for testing)

2. Switch Network if Needed:
   - In wallet settings, change to the correct network
   - Refresh CineX page after switching

Problem 4: Nothing Works on My Browser

What You'll See:
- Wallet extensions won't install
- Or they install but don't work with CineX
- Everything seems broken

How to Fix:
1. Check Your Browser:
   
   These browsers work great:
   - Google Chrome
   - Mozilla Firefox  
   - Microsoft Edge
   - Brave Browser

   These browsers don't support wallets:
   - Safari (Apple's browser)
   - Internet Explorer
   - Most mobile browsers

2. If You're Using Safari or IE:
   - Download Chrome or Firefox
   - Install your wallet there instead


Using CineX on Your Phone

Mobile Users - Different Process:

On Phone/Tablet:
- Browser extensions don't work on mobile
- You need to download wallet apps instead

How to Connect on Mobile:
1. Download Xverse App:
   - iPhone: Get it from App Store
   - Android: Get it from Google Play Store

2. Set Up Your Wallet in the App

3. Access CineX:
   - Open CineX in your mobile browser
   - When you click "Connect Wallet", it should open your Xverse app
   - Approve the connection in the app

Simple Step-by-Step Fix

Complete Setup Guide for New Users:

Step 1: Get the Right Browser
- Download Google Chrome if you don't have it
- (Firefox or Edge also work)

Step 2: Install a Wallet
- Go to Chrome Web Store
- Search "Xverse Wallet"
- Click "Add to Chrome"
- Follow the setup instructions

Step 3: Get Some Test Funds
- If using testnet, get free test STX from faucet
- Ask on CineX community for help with this

Step 4: Connect to CineX
- Refresh the CineX page
- Click "Connect Wallet"
- Choose Xverse from the popup
- Approve the connection

Step 5: You're Ready!
- Your wallet should now show as connected
- You can start using CineX features

Still Having Problems?

Quick Fixes to Try First:

1. The Magic Refresh:
   - Close your browser completely
   - Open it again
   - Go to CineX and try connecting

2. Check Your Wallet:
   - Click your wallet extension icon
   - Make sure it's unlocked (enter password if needed)
   - Check it's on the right network

3. Try Different Wallet:
   - If Xverse doesn't work, try Leather wallet
   - Sometimes one works better than the other

Get Help:
- Contact CineX support with this info:
  - What browser you're using
  - Which wallet you installed  
  - What error message you see (if any)
  - Whether you're on desktop or mobile

Common Solutions:
- 90% of problems: Installing Xverse extension fixes it
- 9% of problems: Switching to Chrome/Firefox fixes it  
- 1% of problems: Need individual troubleshooting help
